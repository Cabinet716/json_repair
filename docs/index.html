JSON Repair Demo

This is a demo page for the Python package json_repair, you can find more information visting the GitHub repository: https://github.com/mangiucugna/json_repair/.

Paste malformed JSON in the left textarea and the repaired JSON will be displayed on the right.

Please note: the api is hosted on pythonanywhere for free. So if too many have used it in a day, it might reach quota and not work
